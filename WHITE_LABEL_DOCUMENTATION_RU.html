Документация

Обзор

Это комплексное eSIM решение, которое предоставляет предприятиям возможность предлагать услуги eSIM своим клиентам. Решение состоит из нескольких компонентов, работающих вместе для предоставления полной платформы управления eSIM.

Компоненты

1. Облачная панель администратора

Расположение: Облачное решение (не предоставляется как саморазмещаемое)

Функции:

  • Бизнес-панель: Обзор бизнес-метрик, доходов и производительности
  • Управление пользователями: Управление учетными записями клиентов и бизнес-пользователей
  • Управление API ключами: Генерация и управление API ключами для бизнес-операций
  • Отслеживание баланса: Мониторинг балансов счетов и истории транзакций
  • Аналитика: Подробная отчетность по продажам eSIM, использованию и поведению клиентов
  • Управление KYC: Обработка процессов верификации "Знай своего клиента"
  • Конфигурация: Управление ключами Stripe, учетными данными Airalo и настройками системы

Доступ: Предоставляется как облачное веб-приложение

2. Облачный API сервер

Расположение: Облачное решение (не предоставляется как саморазмещаемое)

Методы аутентификации:

  1. Аутентификация API ключа: Для бизнес-пользователей
  2. Аутентификация JWT токена: Для обычных клиентов
  3. Интеграция OAuth: Социальный вход через Google и Yandex
  4. Публичные эндпоинты: Для неаутентифицированного доступа

Основные эндпоинты:

Публичные эндпоинты (Без аутентификации):

GET /api/public/countries - Получить доступные страны
GET /api/public/plans - Получить планы eSIM с фильтрацией

Бизнес API эндпоинты (Требуется API ключ):

POST /api/esim/create - Создать eSIM
GET /api/esim/details - Получить детали eSIM
GET /api/esim/usage - Получить статистику использования

Пользовательские эндпоинты (Требуется JWT токен):

GET /api/user/balance - Получить баланс пользователя
POST /api/user/order - Создать заказ eSIM
POST /api/user/qr-code - Получить QR код
POST /api/user/sim-details - Получить детали SIM
POST /api/user/sim-usage - Получить данные об использовании

Функции:

  • Режим Sandbox: Тестовые данные для тестирования
  • Режим Production: Реальные покупки eSIM
  • Управление балансом: Автоматическое списание затрат
  • Отслеживание транзакций: Полный аудит

Доступ: Предоставляется как облачные API эндпоинты

3. Саморазмещаемый Frontend (Next.js)

Репозиторий: /Users/admin/Documents/GitHub/esim-globalbanka/

Технологический стек:

  • Фреймворк: Next.js 14.2.5
  • Стилизация: Tailwind CSS
  • Аутентификация: MongoDB + JWT Auth
  • База данных: MongoDB Atlas (Облачная база данных)
  • Платежи: Интеграция Stripe
  • Интернационализация: i18next (поддержка 7 языков)
Настройка MongoDB: Frontend требует конфигурации MongoDB Atlas для:
  • Подключение к базе данных: Подключение к кластеру MongoDB Atlas
  • Аутентификация: Аутентификация пользователей на основе JWT
  • Хранение данных: Данные пользователей, заказы и информация eSIM
  • Интеграция OAuth: Социальный вход через Google и Yandex

Ключевые функции:

Интерфейс клиента:

  • Поддержка многоязычности: Английский, Испанский, Французский, Немецкий, Арабский, Иврит, Русский
  • Поддержка RTL: Поддержка языков справа налево для арабского и иврита
  • Адаптивный дизайн: Подход дизайна с приоритетом мобильных устройств
  • Выбор страны: Просмотр планов eSIM по странам
  • Сравнение планов: Сравнение различных пакетов eSIM
  • Генерация QR кодов: Мгновенные QR коды для активации eSIM

Панель пользователя:

  • Управление заказами: Просмотр и управление заказами eSIM
  • Отслеживание использования: Мониторинг использования данных и срока действия
  • Настройки аккаунта: Обновление профиля и предпочтений
  • История платежей: Отслеживание платежных транзакций
  • Доступ к QR кодам: Скачивание и просмотр QR кодов для купленных eSIM

Система аутентификации:

  • Регистрация Email/Пароль: Стандартная аутентификация с JWT
  • Верификация Email: Верификация email на основе OTP
  • Сброс пароля: Безопасное восстановление пароля
  • Google OAuth: Интеграция социального входа
  • Yandex OAuth: Социальный вход, удобный для России
  • Управление сессиями: Постоянные пользовательские сессии на основе JWT

4. Саморазмещаемая интеграция платежей (Stripe)

Сервер платежей: https://pay.roamjet.net (Облачная обработка платежей)

Функции:

  • Stripe Checkout: Безопасная обработка платежей
  • Поддержка мультивалютности: USD и другие валюты
  • Тестовые/Продакшн режимы: Sandbox и продакшн среды
  • Интеграция Webhook: Уведомления о платежах в реальном времени
  • Поддержка возвратов: Автоматизированная обработка возвратов

Поток платежей:

  1. Клиент выбирает план eSIM
  2. Frontend создает сессию оплаты
  3. Перенаправление на Stripe Checkout
  4. Обработка платежа
  5. Уведомление Webhook
  6. Создание заказа через API
  7. Генерация QR кода

Развертывание саморазмещаемого Frontend (Next.js)

1. Установка зависимостей:

cd /Users/admin/Documents/GitHub/esim-globalbanka/
npm install

2. Конфигурация окружения:

# Создать .env.local
# Конфигурация MongoDB
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/?retryWrites=true&w=majority

# Конфигурация JWT
JWT_SECRET=your-super-secure-jwt-secret-key-here
JWT_EXPIRES_IN=7d

# Конфигурация Google OAuth
NEXT_PUBLIC_GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret

# Конфигурация Yandex OAuth
NEXT_PUBLIC_YANDEX_APP_ID=your-yandex-app-id
YANDEX_APP_SECRET=your-yandex-app-secret

# Конфигурация API
NEXT_PUBLIC_ROAMJET_API_KEY=rjapi_2k9lt4821123xd7p2dl37mv48jukgo51

# Конфигурация Stripe
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_your-stripe-key
STRIPE_SECRET_KEY=sk_test_your-stripe-secret

# Конфигурация сервиса email
EMAIL_SERVICE_URL=https://smtp.roamjet.net
Настройка MongoDB: Frontend требует конфигурации MongoDB Atlas для:
  • Подключение к базе данных: Подключение к кластеру MongoDB Atlas
  • Аутентификация: Аутентификация пользователей на основе JWT
  • Хранение данных: Данные пользователей, заказы и информация eSIM
  • Интеграция OAuth: Социальный вход через Google и Yandex

3. Сборка и развертывание:

npm run build
npm start

Среда Sandbox

Среда sandbox предоставляет:

  • Без реальных затрат: Все транзакции бесплатны
  • Неограниченный баланс: Идеально для тестирования
  • Мгновенные QR коды: Мгновенный ответ для тестирования

Документация API

Аутентификация

Аутентификация API ключа:

Authorization: Bearer rjapi_2k9lt4821123xd7p2dl37mv48jukgo51

Аутентификация JWT токена:

Authorization: Bearer JWT_TOKEN
X-API-Key: rjapi_2k9lt4821123xd7p2dl37mv48jukgo51

Примеры вызовов API

Получить доступные страны:

curl https://api.roamjet.net/api/public/countries

Получить планы eSIM:

curl "https://api.roamjet.net/api/public/plans?country=FR&limit=10"

Создать заказ eSIM:

curl -X POST https://sandbox.roamjet.net/api/user/order \
  -H "Authorization: Bearer JWT_TOKEN" \
  -H "X-API-Key: rjapi_2k9lt4821123xd7p2dl37mv48jukgo51" \
  -H "Content-Type: application/json" \
  -d '{
    "package_id": "package-id",
    "quantity": "1",
    "to_email": "customer@example.com",
    "description": "Описание заказа"
  }'

Получить QR код:

curl -X POST https://sandbox.roamjet.net/api/user/qr-code \
  -H "Authorization: Bearer JWT_TOKEN" \
  -H "X-API-Key: rjapi_2k9lt4821123xd7p2dl37mv48jukgo51" \
  -H "Content-Type: application/json" \
  -d '{"orderId": "mongodb-order-id"}'
